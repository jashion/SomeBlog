Hexo Blog 搭建（Hexo + Github）

本Hexo版本是3.1.1，在搭建过程中遇到各种坑，被坑的不要不要的，内心是千万只草泥马奔腾而过，废话不多说，下面进入正文。

Node.js

首先，你的电脑必须装有Node.js,Node.js,Node.js（重要的事情要说三遍），毕竟 Hexo开源Blog是居于Node.js的环境搭建的，至于怎么安装Node.js网上百度一大把的教程，在这就不重点描述。

Github

关于这个Github账号，大家都知道，你的Blog搭建完成是需要一个服务端的，而Github就可以免费充当这个服务端，所以，没有Github账号的同学必须申请一个Github账号，这个很重要。
（1）创建Github账号，这个省略1000字
（2）新建一个公共的Repository(仓库)，改仓库的名字有讲究，貌似必须要这样的格式(具体原因不明，应该是Hexo/Github规定的格式)：
your_username.github.io //your_username替换成个人的github名字，我的是jashion.github.io ，大家可以多多交流
（3）检查SSH keys的设置
check ssh key:

如果提示：No such file or directory 则需要新建一个SSH Key

new SSH Key:

然后会提示输入钥匙串加密密码，这个密码是你提交项目的用到的验证，防止别人随便篡改你的Blog，可以不输入。（温习提示：最好创建一个密码，然后记住它）
创建成功后会出现一个虚线的方框，这里忘记截图了，请原谅我一时忘了截图。
（4）添加SSH Key到Github

  • 1: 打开本地C:\Documents and Settings\Administrator.ssh\id_rsa.pub文件

  • 2: 复制该文件的内容，然后登录Github，选择如下图：
    (1)选择setting

    (2)选择SSH keys

    (3)Title随便写，key就是1中的文件内容

  • 3: 测试是否关联Github账号成功

会出现一些提示语句，不用太过担心，只要回答yes就好。
